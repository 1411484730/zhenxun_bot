# 绪山真寻Bot
****
此项目基于 Nonebot2 和 go-cqhttp 开发的QQ群娱乐机器人
## 关于
用爱发电的项目，某些功能借鉴了大佬们的代码，因为绪山真寻实在太可爱了因此开发了 
绪山真寻bot，实现了一些对群友的娱乐功能和实用功能（大概）。

## 声明
此项目仅用于学习交流，请勿用于非法用途

## 真寻的帮助
请对真寻说: '真寻帮助' or '管理员帮助' or '超级用户帮助' or '对应指令 帮助'
